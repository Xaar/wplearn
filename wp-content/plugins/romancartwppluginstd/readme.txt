Roman Cart Wordpress Plugin Std easily link Wordpress to a RomanCart shopping cart no html required!

== Description ==

Provides shortcodes that can easily be included in page or post text to simplify
integration with the RomanCart website.

Thanks to Steve Whiteley for the basic plugin template

== Installation ==

1. Upload the `RomanCartWPPluginStd` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go to the Admin -> Settings -> RomanCart page and set your RomanCart StoreId.
1. use the [romancart] shortcodes in your pages and posts as outlined on the settings page

== Frequently Asked Questions ==

= Where do I get a RomanCart StoreId (I already have a RomanCart account)? =

You have to enter the StoreId whenever you log into the RomanCart website.
It's the exact same number that you enter into the settings page for this plugin.

= Where do I get a RomanCart StoreId (I do not have a RomanCart account)? =

*You can signup for a RomanCart account for free* (as at the time of writing - Apr 2010)
using the following url:
   [RomanCart Registration](http://www.romancart.com/register.asp)
